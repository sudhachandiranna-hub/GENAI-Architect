🧭 Choosing the Right Framework
RTCFR General Content
Best for text-based content — like a salesperson's pitch or marketing copy. Good general-purpose structure.
SCALE Advanced
A different, more advanced framework — worth knowing exists. Full details will be covered in depth later.

For image and video generation specifically, use JSON prompting — not RTCFR or SCALE. JSON gives you a much higher level of structured detail than plain text ever can, which is exactly what visual generation needs.

📝 Markdown Prompt Format

Another format worth knowing: the Markdown prompt — used for generating Markdown files, along with a set of extra expressions/syntax specific to this format. With this, the core basics of prompting are now covered.
